# Instale as dependências
npm install

# Iniciar o projeto
npx expo start

# Ou use um dos comandos específicos:
npx expo start --android  # Para Android
npx expo start --ios      # Para iOS (apenas Mac)
npx expo start --web      # Para Web

Testar no Celular

Instale o app Expo Go no seu celular

- [Android - Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)
- [iOS - App Store](https://apps.apple.com/app/expo-go/id982107779)


Execute npx expo start
Escaneie o QR Code:

Android: Use o app Expo Go
iOS: Use a câmera nativa do iPhone
